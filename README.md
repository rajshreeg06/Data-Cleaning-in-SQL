# Data Cleaning & Analysis using SQL

## Project Overview
This project focuses on cleaning and analyzing housing data using SQL. The dataset was processed to improve data quality and prepare it for analysis and reporting.

## Tools Used
- SQL (T-SQL)
- Microsoft Excel
- Power BI (optional)

## Key Tasks
- Removed duplicates
- Handled missing values
- Standardized date formats
- Split address into multiple columns
- Performed data analysis

## Dataset
Nashville Housing Dataset

## Outcome
Improved data quality and generated insights to support data-driven decisions.
